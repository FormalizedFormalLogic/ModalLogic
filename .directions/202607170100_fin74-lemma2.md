# Fin74 Lemma 2 stage 2: `validates_G` の進捗

対象: `Fin74/Lemma2/ValidatesG.lean`．目標定理は
`theorem validates_G : Fin74.Lemma2.frame.Validates Fin74.G`（`G = E 🡒 F`）．
これは**任意の付値 `ψ`**（`Model.lean` の固定 `φ` ではない）に対し，`w ⊩ E → w ⊩ F` を示すもの．

## 完成した部分（`sorry` なし）

- **`exists_V_shape`**: 論文の「(W,R) restricted to {w,u₀,v₀,u₁,v₁} が V字型」ステップを厳密化した補題．
  任意の `ψ` の下で `w ⊩ E` から次を取り出す．
  - `R w t`, `t ⊩ A 0`（`= ◇B₁ ⋏ ◇C₁ ⋏ ∼◇B₂`）
  - `R t u₁`, `R u₁ u₀`, `R t v₁`, `R v₁ v₀`
  - `u₁ ⊩ B 1`, `u₀ ⊩ B 0`, `v₁ ⊩ C 1`, `v₀ ⊩ C 0`
  - 4つの非到達性 `¬R u₀ u₁`, `¬R v₀ v₁`, `¬R u₁ v₀`, `¬R v₁ u₀`
  導出は `forces_E_iff` で `D ∧ ◇A0 ∧ K 0` に分解し，`◇A0` から `t`，その中の `◇B1`/`◇C1` から
  `u₁`/`v₁`，`K1 0`/`K2 0` の box を `u₁`/`v₁` に適用して `u₀`/`v₀` と `∼◇C0`/`∼◇B0`，
  `K3 0`/`K4 0` を `u₀`/`v₀` に適用して `∼◇B1`/`∼◇C1` を得て，非到達性を出す．論文の step 3 に対応．
  `φ` は一切使わず `R` の構造と `E` の様相的内容だけを使う（＝frame validity として正しく arbitrary `ψ`）．

- **構造補題群（`R` の場合分けのみ，全て真）**:
  - `pred_a` / `pred_a_index`: `a`点に到達できるのは `a`点・`d`点のみ（`b`/`c`点からは到達不可）．
  - `R_d_a` / `R_d_d` / `R_d_b` / `R_d_c`: `d`点からの到達条件．
  - `not_R_a_d`: `a`点は `d`点に到達できない．
  最終ステップの `F`-witness `d_{n+1}` の構成（`d_{n+1} ≺ a_{n+1}`, `d_m ≺ d_{n+1}`）と，
  `w` が `a`点でないことの絞り込みに使う想定．

## `sorry` が残っている場所

- **`validates_G` の本体**（`ValidatesG.lean` 内）．`exists_V_shape` で V字型を取り出した後の
  **組合せ論的コア**が未形式化．やるべきは:
  1. V字型（R-関係＋forcing）から，ある `n > 0` について `t = a_n`, `u_i = b_{n+i}`,
     `v_i = c_{n+i}`, かつ `w = d_m`（`m ≤ n`）を導く分類．論文の "it is now easy to verify" の部分．
  2. `F`-witness として `d_{n+1}` を構成: `d_{n+1} ≺ a_{n+1}` で `◇(A 1)`，`d_{n+1} ∦ a_n` で
     `∼◇(A 0)`，`w = d_m ≺ d_{n+1}` と `w ⊩ D` で `p₀ ⋎ p₁`．よって `w ⊩ F`．

## 詰まった箇所・重要な発見

- **論文の "w ⊩ D だけで w = d_m" は arbitrary `ψ` の下では偽**．反例: `p₁` を全点で真にする `ψ` を
  取ると，`D` の第2連言 `□(p0 → …)` は前件が空で自明，第3連言 `□(∼(p0∨p1) → …)` も `∼(p0∨p1)` が
  どこでも偽で自明，第1連言 `p0∨p1` は `p₁` で成立．ゆえに `D` は cone が有限な `a`/`b`/`c`点でも
  成立する（例: `b_0` の cone は `{b_0}`）．したがって `w` が `d`点であることは `D` 単独ではなく
  `E` 全体（分類）から出さねばならない．タスクの進め方の提案 step 2（`w ⊩ D → ∃m, w = .d m`）は
  このままでは証明不能なので採用しなかった．

- 分類が難しい本質的理由: `B i`/`C i` のうち `B 0 = q0`, `B 1 = q1`, `C 0 = r0`, `C 1 = r1` は
  ただの原子（`ψ` で任意の点で真にできる）なので，`u₁ ⊩ B 1` から `u₁` の型（`b`点か等）は直接出ない．
  型の同定は `t ⊩ ∼◇B₂` などの**合成論理式**（`B 2 = ◇B1 ⋏ ◇C0 ⋏ ∼◇C1`）が与える再帰的な様相制約と
  `K`ブロック，および `R` の構造の組合せから出る．論文が「B_i and C_i are true at none other of the
  b's or c's」と言う主張は `i` についての帰納法を要し，これがこのプロジェクト最難関のコア．

## 次にやるべきこと

1. まず `w` の型の絞り込み: `t = a_n` を先に確定できれば `pred_a` で `w ∈ {a点, d点}`．
   `w = a点` の場合（`R a_m a_n ⟺ n = m` より `w = t = a_n`）は `E` の残りの強さ（`◇A0` の witness が
   さらに無限昇鎖を要求する等）で排除する必要がある．ここが未検討で，本当に排除できるか（＝
   `a`点で `E` 全体が成立し得ないか）は要確認．成立し得るなら定理自体の見直しが必要になるが，
   Fine の結果は frame validity を主張しているので排除できるはず．
2. 分類の帰納補題（`B i`/`C i` がどの点で forcing され得るかの一般化）を，`R` の strict order 構造
   （`b`/`c` は index 降下，`d` は昇鎖，`a` は孤立）を使って立てる．
3. `d_{n+1}` witness 構成は構造補題群が既にあるので，分類さえ済めば比較的短いはず．
